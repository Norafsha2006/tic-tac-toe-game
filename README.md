# 🎮 Tic-Tac-Toe Game

A modern Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features

- Player vs Player mode
- Player vs AI mode
- Dark / Light theme toggle
- Score tracking
- Match reset and new game options
- Responsive design for mobile and desktop
- Local storage support for scores

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)


## 🎯 How to Play

1. Choose a game mode.
2. Click on any empty cell to make a move.
3. Get three marks in a row, column, or diagonal to win.
4. Track scores and start a new match anytime.
